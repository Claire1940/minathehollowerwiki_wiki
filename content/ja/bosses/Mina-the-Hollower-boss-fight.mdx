<Section icon={Swords} title="ボス戦におけるコア戦闘メカニクス">

**Mina the Hollower ボス戦**で成功するには、魅力的な8ビットのビジュアルだけでなく、ミナの動きの技術的な深さをマスターする必要があります。従来のA-RPGとは異なり、このゲームの戦闘は垂直方向と水平方向の両方で行われ、地面そのものを防御ツールとして利用します。「ホロウワー」のスキルセットにより、ミナは地中に潜ることができ、一時的な無敵と再配置の機会を得ることができ、これはゲーム後半の生存に不可欠です。

| メカニクス | 入力アクション | ボス戦での有用性 |
| :--- | :--- | :--- |
| **潜行（Burrow）** | ジャンプ後、潜る | 突進攻撃/投擲物を回避 |
| **ナイトスターウィップ** | 主攻撃 | 中距離での突きと安全な間合い |
| **サイドアーム使用** | 副次的なツール | バーストダメージまたは群衆制御 |
| **跳躍（Vaulting）** | 潜行から浮上 | 間合いを詰めたり、低い掃討攻撃を飛び越えたりする |
| **武器交換** | インベントリ/クイックメニュー | ボスの耐性/速度に適応する |

<Callout type="warning" title="潜行タイミングの警告">
潜行アニメーションにはわずかな開始遅延があります。ボスがすでに突進している最中に潜行しようとすると、ダメージを受ける可能性が高いです。序盤のダンジョンにいる突進してくる敵のような高速な脅威を回避するには、先制的な潜行が唯一の確実な方法です。
</Callout>

ゲーム内の移動にはわずかな加速カーブがあります。一部のプレイヤーは「ボートを操作しているようだ」と表現しますが、この慣性はコミットした動きに報いるように設計されています。ボスアリーナでは、立ち止まることは死を意味します。ボスが画面の端に押し付けないように、常に動き続ける必要があります。

</Section>

<Section icon={Target} title="序盤のボス攻略">

ゲーム序盤の障害は、潜行と間合いの習得度を試す「スキルチェック」として機能します。序盤の体験を特徴づける2つの主要な遭遇戦は、**ワニボス**と**ビッグサグ**サブボスです。これらの戦いでは、ダメージを耐えることは可能ですが、精密な操作が勝利への意図された道であることを教えてくれます。

### ワニボス攻略
ワニボスは、重い物理的な掃討攻撃と噛みつき突進を使用します。その巨体はナイトスターウィップの簡単なターゲットになりますが、そのリーチも同様に印象的です。

| フェーズ | ボスの行動 | 対抗戦略 |
| :--- | :--- | :--- |
| **フェーズ1** | 遅い尻尾の掃討攻撃 | 尻尾の軌道の下をジャンプまたは潜行する |
| **フェーズ2** | 素早い噛みつき突進 | 潜行して突進に対して垂直に移動する |
| **フェーズ3** | 地面叩きつけ/破片 | 砂埃が落ち着くまで潜行したままにする |

<Steps>
  <Step num="1" title="最初の交戦">
    ナイトスターウィップを使って、最大射程から2～3ヒット与えます。ワニボスは素早く反撃してくるので、深追いしすぎないようにしましょう。
  </Step>
  <Step num="2" title="突進の管理">
    ワニが頭を後ろに引いたら、すぐに潜行します。地上で突進を逃れようとしないでください。
  </Step>
  <Step num="3" title="サイドアームバースト">
    地面叩きつけ後のクールダウン中に、サイドアーム（ブーメランソードなど）を使用して、大量のバーストダメージを与えます。
  </Step>
</Steps>

<Callout type="tip" title="ビッグサグのショートカット">
**ビッグサグ**サブボスは、その広範囲の振り回し攻撃のため、回避するのが非常に難しいことで知られています。一部のプレイヤーは、体力アップグレードを少なくとも1つ見つけていれば、彼の攻撃を「耐えながら」、ウィップで最大DPSを与える方が効率的だと感じています。しかし、ノーダメージクリアを目指す場合は、彼が腕をピクッと動かした瞬間に逃げなければなりません。
</Callout>

</Section>

<Section icon={Map} title="中盤の課題：クラーケン">

クラーケンは、最初の主要な環境型ボス戦です。ワニとは異なり、クラーケンは水域の危険と複数の触手によって移動を制限します。この戦いは、純粋なダメージよりもアリーナ管理が重要です。

**クラーケンの主要戦術：**
- **触手の優先順位**: クラーケンの頭部に集中する前に、常に活動中の触手を排除してください。
- **橋での潜行**: 注意してください。一部の橋は幅が1タイルしかありません。ここで潜行しようとすると、落下したり、潜行に失敗したりして、クラーケンのインク攻撃に無防備になる可能性があります。
- **予兆のある掃討攻撃**: クラーケンは掃討攻撃の前に触手を高く上げます。これは潜行して「フーッ」という効果音がするまで待ってから浮上する合図です。

<Callout type="info" title="環境認識">
クラーケンアリーナには破壊可能な足場があります。一箇所に長く留まりすぎると、クラーケンが床を破壊し、ミナの移動速度が著しく遅くなり、潜行できない水中に強制的に落ちてしまいます。
</Callout>

</Section>
