<Section icon={Shield} title="純粋なパッシブ取引品：必須のステータスブースター">

シンプルなアプローチを好むプレイヤーにとって、純粋なパッシブ取引品はミナの基本属性に直接的なボーナスを与えてくれます。これらは、穴掘り回避のタイミングをまだ習得中のゲーム序盤において、最も信頼できるアイテムとなるでしょう。主に攻撃力、防御力、サイドアーマー（副兵装）の有効性に焦点を当てています。

| 取引品名 | 主な効果 | おすすめの用途 |
| :--- | :--- | :--- |
| **レースのグローブ** | 攻撃力を1段階上昇させる | ムチ特化ビルド |
| **ツイルウィード** | 防御力を1段階上昇させる | 初心者 / ボス戦 |
| **気付け薬** | サイドアーマーの攻撃力を上昇させる | 遠距離 / 戦術的プレイ |
| **薬瓶ポーチ** | プラズマ瓶を2つ多く所持できる | 高難易度エリア |
| **スパークオーブ** | スパークスロットを1つ追加する | リソースの保護 |

爆発を引き起こすアイテムに比べるとステータスブースターは「地味」に見えるかもしれませんが、後半のダンジョンの厳しい数値的課題を克服するためには不可欠です。特に**薬瓶ポーチ**は、ミナの回復システムが『Bloodborne』にインスパイアされており、リソース維持に攻撃的なプレイが求められるため、トップクラスの選択肢となります。

<Callout type="tip" title="スタッキングの利点">
スパークオーブを軽視してはいけません。スパークを回収できずに2回死ぬと所持していた骨をすべて失うこのゲームにおいて、わずかなダメージ増加よりも予備のセーフティネットを持つことの方が価値がある場合が多いのです。
</Callout>

</Section>

<Section icon={Sparkles} title="特殊パッシブ：ゲームの流れを変える">

特殊パッシブこそ、**Mina the Hollowerの取引品**システムが真に輝く場所です。これらのアイテムは単にミナを強化するだけでなく、移動、無敵時間、ダメージを受けた際の反応などを変化させます。これらは「ホロウイング（穴掘り）」スキルを磨きたいプレイヤーにとって不可欠です。

  <Card title="回避の粉">

  </Card>
  <Card title="鉄の肺">
    **最大潜行時間**を増加させます。これは長い溝を越えたり、地上からの飛び道具をやり過ごしたりする際に重要です。

  <Card title="不動の魂">
    被弾時の**ノックバック距離**を短縮し、穴やトゲに突き落とされるストレスを軽減します。
  </Card>

これらの取引品は組み合わせることで特に威力を発揮します。例えば、**鉄の肺**と**滲み出る器官**（潜行中にダメージを与えるトレイルを残す）を組み合わせれば、防御的な機動が強力な攻撃手段へと変わります。

| ユーティリティ取引品 | メカニクスの変化 | 戦略のヒント |
| :--- | :--- | :--- |
| **俊敏の薬** | 攻撃後の移動速度が上昇 | ヒット・アンド・アウェイ |
| **マンドレの根** | 叫び声で周囲の敵を吹き飛ばす | クラウドコントロール |
| **血圧のボンネット** | 低体力時にダメージが増加 | ハイリスク・ハイリターン |
| **滲み出る器官** | ダメージを与える粘液の跡を残す | パッシブ・ダメージ |
| **ンダーのベルト** | 地上に飛び出す際に周囲に爆発を起こす | 攻撃的な潜行 |

<Callout type="warning" title="潜行のタイミング">
鉄の肺で長時間潜っていられますが、敵の当たり判定の中に飛び出すとダメージを受けてしまいます。攻撃的な浮上用取引品を装備していない限り注意が必要です。
</Callout>

<Section icon={Swords} title="アクティブ取引品と戦闘アビリティ">

アクティブ取引品は、手動での入力や特定の条件によって強力な効果を発動します。これらのアイテムは、ミナにクラシックな『ゼルダの伝説』や『悪魔城ドラキュラ』を彷彿とさせるアビリティを授けます。これらのアイテムを使いこなせるかどうかが、初心者のホロワーとマスターの分かれ道です。

<Steps>
  <Step num="1" title="コンボを組み立てる">
    **コンボキャパシター**を装備しましょう。攻撃を連続で当てるほどダメージが増加します。ただし、攻撃を空振りするとボーナスがリセットされるため注意が必要です。
  </Step>
  <Step num="2" title="エアダッシュを極める">
    **ふいごのバッスル**を使用してエアダッシュを行います。これは距離を詰める戦闘ツールであるだけでなく、暗黒の島の高所に隠された貴重な宝箱に到達するためにも不可欠です。
  </Step>
  <Step num="3" title="地震パルスを発動する">
    潜行中に**地震のベルト**を使用すると、攻撃ボタンでダメージを与えるパルスを放出できます。地上に出る前に、クモやコウモリなどの小さな敵を一掃するのに最適です。
  </Step>
  <Step num="4" title="空中攻撃を実行する">
    **宙返りのチュチュ**は空中でのセカンダリアタックを可能にします。ムチの射程距離と組み合わせることで、上空から安全にダメージを与え続けることができます。
  </Step>
</Steps>

<Callout type="success" title="探索のシナジー">
**フェアリング・ウィスプ**（ゆっくり落下）と**ふいごのバッスル**（エアダッシュ）を組み合わせれば、プラットフォームセクションを丸ごとスキップできることもあります。これはスピードランナーや隠し要素をコンプリートしたいプレイヤーに人気のコンボです。
</Callout>

</Section>

<Section icon={Target} title="骨の管理と通貨の変換">

最高の**Mina the Hollower取引品**を入手するには、安定した骨の供給が必要です。このゲームは『リンクの冒険』を彷彿とさせるレベリングシステムを採用しており、経験値がしきい値に達すると体力、攻撃力、魔法（サイドアーマー）のいずれかをアップグレードできます。しかし、Yacht Club Gamesはここに素晴らしいひねりを加えました。それが**骨石の破片（Bonestone Shards）**です。

| 通貨の種類 | 永続性 | 主な用途 |
| :--- | :--- | :--- |
| **骨** | 死亡時に紛失（スパーク回収失敗時） | ステータスのレベリング / 取引品の購入 |
| **骨石の破片** | 死亡後も**維持される** | 高価なアイテムのための安全な長期貯蓄 |
| **スパーク** | 死亡地点で回収可能 | 現在保持している骨の保護 |

レベルアップ時、現在の骨の蓄積状況を骨石の破片に変換するオプションがあります。これはボス戦や高リスクエリアに挑む前の重要な戦略です。これにより、苦労して稼いだ進捗が一つのミスで台無しになるのを防ぐことができます。
