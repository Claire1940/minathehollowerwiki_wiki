このゲームのダメージアップグレードは普遍的です。攻撃力を上げるためにリソースを投資すると、そのブーストは、見つけた、または鍛冶屋から購入したすべての武器に適用されます。これにより、後でプレイスタイルを変更してもペナルティなしで実験が奨励されます。

| 武器タイプ | 理想的なシナリオ | 初心者評価 |

| **ブラストストライクモール** | 遅い装甲エリートを罰する | ★★★☆☆ |
| **ウィスパー＆ヴェスパー** | ヒットボックスの大きいボス | ★★☆☆☆ |
| **サイドアーム** | 特定の部屋の危険を解決する | ★★★★☆ |

<Section icon={Sparkles} title="穴掘りとサイドアームの力">

穴掘りはミナの移動能力の核心です。穴掘りボタンを押すと、ミナは地下に潜り込み、ほとんどの地表攻撃に対して無敵になります。これがあなたの主要な回避手段です。多くの新規プレイヤーは、穴掘りを隙間を渡る手段としてのみ扱って苦戦しますが、実際には最高の防御層です。

**穴掘りを効果的に使う方法:**
1.  **突進の予測**: 敵が予備動作を開始したら、すぐに潜り込みましょう。
2.  **影の追跡**: 地下では、影があなたの真の位置を示します。「飛び出し」攻撃を仕掛けるために利用しましょう。
3.  **ショートカット探索**: 小さな亀裂のあるフェンスや壁を探しましょう。その下を潜り抜けるのが、しばしば意図された道です。

<Callout type="info" title="サイドアームとジュール">
サイドアーム（ボルトハチェットやジャイロダガーなど）は**ジュール**を消費します。これらは強力ですが、限りがあります。メイン武器とは異なり、サイドアームは**死ぬと失われます**。「後で」のために貯め込まず、戦利品を守る困難なエリート敵を排除するために使いましょう。
</Callout>

| リソース | 入手方法 | 使用方法 |
| :--- | :--- | :--- |
| **骨** | 敵を倒す、壺を壊す | レベルアップとアイテム購入 |
| **ジュール** | オブジェクトを壊す、ジュールジャー | サイドアームの動力源 |
| **スパーク** | チェックポイント、主要なマイルストーン | 死後の回復試行回数 |
| **装飾品** | 探索、サイドクエスト | パッシブバフと復活 |

</Section>

<Section icon={Shield} title="死、骨、そしてプロトスパーク">

他のソウルライクな体験と同様に、『Mina the Hollower』では、死ぬと現在所持している**骨**を落とします。骨を回収できるチャンスは限られています（スパークによって決定されます）。死亡現場にたどり着く前に再度死んでしまうと、その骨は消滅します。

しかし、このゲームは見た目よりも寛大です。ほとんどの地域は密集しており、ルートが分かれば、敵を無視し、画面遷移を利用してアグロをリセットすることで、1分以内に死亡場所まで走り戻ることができます。

<Steps>
  <Step num="1" title="墓地のクエスト">
    墓地地域の終わり近くで、「ゴーストデュード」NPCを見つけましょう。
  </Step>
  <Step num="2" title="護衛ミッション">
    ボス戦直後にある施錠された部屋まで彼を護衛しましょう。
  </Step>
  <Step num="3" title="プロトスパークを手に入れよう">
    部屋の中で、初心者にとって最も不可欠な装飾品である**プロトスパーク**を受け取ります。
  </Step>
  <Step num="4" title="生存のために装備しよう">
    この装飾品は、チェックポイントごとに1回、部分的な体力であなたを復活させます。ボス戦ではゲームチェンジャーとなります。
  </Step>
</Steps>

<Callout type="success" title="消費戦略">
貯め込んではいけません。店に着いたらすぐに、永続的な体力アップグレード（体力+25%）や鍵に骨を使いましょう。わずかなダメージブーストは、ボスからの一撃を余分に耐えることよりも価値が低いです。
</Callout>

</Section>

<Section icon={Target} title="進行と店の優先順位">

テネブロス島はセミオープンワールドです。もし「壁に頭を打ち付けている」と感じたら、「勇敢に後退する」方が良いことがよくあります。別の地域を探索し、新しい装飾品を見つけ、より装備が整ってから戻りましょう。

| 店のアイテム | 優先度 | 理由 |
| :--- | :--- | :--- |
| **体力アップグレード** | **高** | 戦闘でのミスの許容範囲を広げます。 |
| **鍵** | **高** | ショートカットや価値の高い宝箱を開けます。 |
| **新しい武器** | **中** | 自分の手に合ったスタイルを見つけるのに役立ちます。 |
| **ダメージ +1** | **低** | 高価であり、序盤の生存能力よりも影響が少ないです。 |

「骨アップ」ステーションでレベルアップする際は、自分の弱点に焦点を当てましょう。ボス戦が長引きすぎると感じるなら、攻撃に投資しましょう。3発で死んでしまうなら、防御を優先しましょう。装飾品システム（60種類以上見つかります）の柔軟性により、セーブを最初からやり直すことなくビルドを変更できます。

<Callout type="tip" title="秘密の探索">
部屋に明らかな出口がない場合や、アイテムが届かないように見える場合は、壁に穴を掘ってみましょう。開発者は、真のホロウワーだけが見つけられる「空洞の」壁の裏に、レベル全体を隠しています。
</Callout>

</Section>
