<Section icon={Map} title="ホローイングと移動メカニクスの習得">

テネブラス島での冒険の核心は、ミナのシグネチャーアクションであるホローイングに関する**Mina the Hollowerのヒント**にあります。ジャンプや隙間を渡るのに特定のアイテムが必要な従来のトップダウンアドベンチャーゲームとは異なり、ミナは自由に地面に潜り込むことができます。この「ホローイング」は、スピード、回避、パズル解決のための主要なツールとして機能します。潜り込むと、通常の歩行速度よりもかなり速く移動できますが、地面に出入りするのに必要なタイミングを意識する必要があります。

**ビデオハイライト：**
- 3つの初期武器（ナイトスター、マリス、ウィスパー/ヴェスパー）の詳細な解説。
- 環境カバーを利用して最初の主要ボスを倒すためのステップバイステップの攻略法。
- ソーンとのボス戦の戦闘戦略、攻撃のタイミングを重視。
- 積極的な近接攻撃による体力回復システムの解説。

ホローイングは短い無敵時間を提供し、敵の投射物や突進攻撃の下に潜り込むことができます。ただし、地面から現れるアニメーション中は一時的に無防備になります。これにより、スピードアップが再浮上時の捕獲リスクに見合うかどうかを判断しなければならない、ハイリスクな「ミクロ意思決定」ループが生まれます。

| メカニクス | 主な用途 | 戦略的利点 |
| :--- | :--- | :--- |
| 歩行 | 精密な移動 | アニメーションロックを回避し、慎重な間合いを可能にする |
| ホローイング | スピードと隙間 | 穴を渡ったり、突進を回避したりするのに不可欠 |
| 出現 | 攻撃のセットアップ | シールドを持った敵の背後に現れるために使用できる |
| ボーン収集 | 進行 | ステータスをレベルアップさせたり、重要な装飾品を購入したりするために使用 |

<Callout type="tip" title="潜り込み戦略">
障害物や動かない番兵に遭遇した場合、常に最初にホローイングを試みるべきです。多くの初期の「ソフトロック」は、実際には単純な潜り込みパズルです。障害物の下に道があることを示す柔らかい土や亀裂を探してください。
</Callout>

<Section icon={Sparkles} title="ボーンアップ進行システム">

Mina the Hollowerの進行は、「ボーンアップ」システムを利用しており、伝統的なRPGのレベルアップと通貨管理が独自に融合しています。敵を倒したり、地面を掘り進んだりすると、ボーンを収集します。これらは経験値として機能します。特定の閾値に達すると、コア属性の1つをレベルアップさせることができます。

<Steps>
  <Step num="1" title="ボーンの蓄積">
    敵を倒し、柔らかい場所を掘ってボーンを集めましょう。序盤では、戦闘よりも掘削の方が儲かることがよくあります。
  </Step>
  <Step num="2" title="ステータスの選択">
    攻撃力、防御力、敏捷性の中から選択します。攻撃力は純粋なダメージを増加させ、防御力は死亡時の「ボーンロス」ペナルティを減少させます。
  </Step>
  <Step num="3" title="装飾品への投資">
    ボーンの一部をNPCから装飾品を購入するために温存しましょう。これらは、潜り込み時間の延長や体力回復の増加などのパッシブバフを提供します。
  </Step>
  <Step num="4" title="死亡からの回復">
    戦闘で倒れると、ボーンの山が残されます。その場所に戻って進行状況を取り戻す必要があります。これはソウルライクなメカニクスに似ています。
  </Step>
</Steps>

| ステータスカテゴリ | 利点 | 推奨ビルド |
| :--- | :--- | :--- |
| 攻撃力 | 一撃あたりのダメージ増加 | 「グラスキャノン」（攻撃力/敏捷性に特化） |
| 防御力 | 受けるダメージの減少 | 「タンク」（防御力/体力に特化） |
| 敏捷性 | ホローイングと回復の高速化 | 「スピードスター」（敏捷性/装飾品に特化） |

<Callout type="success" title="体力管理">
敵を攻撃すると体力回復ポーションが補充されることを忘れないでください。体力が少ない場合は、ただ逃げるのではなく、動きの遅い敵を見つけて数回攻撃し、体力を回復しましょう。これは、長いダンジョンを生き残るための**Mina the Hollowerのヒント**の中で最も重要です。
</Callout>

</Section>

<Section icon={Trophy} title="序盤のボス戦略：モグラとソーン">

最初の2回のボス戦は、ホローイングと間合いの理解度を試すスキルチェックとなります。最初のボス、巨大な地下生物は、環境を有利に利用する必要があります。ボスが突進攻撃を準備しているときは、破壊不可能な岩の形成物の後ろに隠れてください。壁にぶつかった後の回復の隙を狙って、サブウェポンを放ちましょう。

2番目のボス、ソーンとの戦いでは、難易度が大幅に上昇します。ソーンは、高機動の槍攻撃と火の竜巻を使うライバルの戦士です。ここでの鍵は忍耐です。

| ボスフェーズ | ソーンの攻撃 | 対抗策 |
| :--- | :--- | :--- |
| 空中ジャンプ | 上から叩きつける | 影を待ち、潜り込んで回避し、2回攻撃 |
| 火の竜巻 | 包囲 | 中央に留まるか、炎の下に潜り込む |
| 槍突き | 長距離突き | 横に避け、斧のようなサブウェポンを使用 |
| 絶望 | 連続する波 | 無敵時間のためにホローイングに集中 |

<Callout type="tip" title="ソーンとの戦闘ウィンドウ">
ソーンは、ジャンプから着地した後の約2秒間しか無防備になりません。欲張らないでください。鞭やメイスで2回攻撃し、すぐに潜り込んで反撃を避けましょう。
</Callout>

</Section>
