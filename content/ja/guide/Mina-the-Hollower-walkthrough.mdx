**Mina the Hollower ウォークスルー**の序盤で適切な武器を選ぶことが、あなたの生存率を決定します。初心者には、**ナイトスター**が優れた選択肢です。その射程により、危険な範囲に入ることなく敵の予備動作を観察できます。

| 戦闘行動 | 主な用途 | エキスパートアドバイス |
| :--- | :--- | :--- |
| **攻撃** | ダメージを与える | 1～2回攻撃し、位置を再調整する |
| **回避** | 素早い回避 | 隙間を詰めたり、範囲攻撃から逃れるために使用する |
| **潜行** | 無敵 | 危険な場所の下に無期限に留まるために保持する |
| **サイドアーム** | ユーティリティ / 遠距離 | 飛行する害虫には**ボルトハチェット**を使用する |

<Callout type="warning" title="戦闘の警告">
攻撃ボタンを連打しないでください。ほとんどの敵には攻撃後に回復時間があります。ヒットを当てたら、すぐに回避または潜行して位置をリセットしてください。
</Callout>

<Section icon={Sparkles} title="進行：ボーン、ジュール、装飾品">

Mina the Hollowerの進行は、ソウルライクゲームと同様の「リスクと報酬」の構造に従います。主要な通貨は**ボーン**で、「ボーンアップ」（レベルアップ）してステータスを上げます。未知のエリアに大量のボーンを持ち込んで死亡すると、蓄積した財産を失うリスクがあります。

<Steps>
  <Step num="1" title="ボーンを集める">
    敵を倒したり、オブジェクトを破壊したりしてボーンを集めます。あまり長く溜め込まないようにしましょう。
  </Step>
  <Step num="2" title="ボーンアップステーションを見つける">
    チェックポイントを見つけて、ボーンを攻撃、防御、またはサイドアームのステータスに投資します。
  </Step>
  <Step num="3" title="装飾品を装備する">
    受動的なバフを追加する装飾品を見つけるか購入します。合計60種類を発見できます。
  </Step>
  <Step num="4" title="ジュールを管理する">
    サイドアームに電力を供給するためにジュールを集めます。覚えておいてください：サイドアームは死亡時に失われます。
  </Step>
</Steps>

どのステータスを優先するかを決定する際は、最近の死亡状況を評価してください。戦闘が長引き、疲労のためにミスを犯している場合は、**攻撃力**を上げてください。基本的なザコ敵に2発で倒されている場合は、**防御力**に重点的に投資してください。

| 問題 | 推奨ステータス | 期待される効果 |
| :--- | :--- | :--- |
| **頻繁な死亡** | 防御力 | 生存率と学習時間の増加 |
| **クリアが遅い** | 攻撃力 | 敵のサイクルが減り、ボスを早く倒せる |
| **資源不足** | サイドアーム | より良いユーティリティと遠距離オプション |
| **ポジショニングエラー** | 装飾品 | 回避または潜行速度のカスタムバフ |

<Callout type="success" title="ビルド戦略">
装飾品はビルドを柔軟にします。特定のボスに苦戦している場合は、装飾品を交換して、そのボスの特定のダメージタイプや動きのパターンに対抗しましょう。一度の死亡でビルド全体を変更するのではなく、効果をテストするために一度に1つの装飾品を変更してください。
</Callout>

</Section>

<Section icon={Target} title="6つのスパークジェネレーターのウォークスルー">

あらゆる**Mina the Hollower ウォークスルー**の核となるのは、6つのスパークジェネレーターへの旅です。これらはゲームの主要なマイルストーンとして機能します。テネブラス島は非線形であり、どのルートを最初に攻略するかを自由に選択できます。しかし、あるルートが消耗しすぎると感じた場合は、「悪い壁」を無理に進むよりも、別の道を探しに行く方が良いでしょう。

1.  **中距離で新しい部屋に入る**: 中央に突入しないでください。
2.  **まず観察する**: 敵がどのように動き、どのような環境の危険があるかを確認します。
3.  **脅威を排除する**: まず遠距離攻撃の敵に集中し、次に近接攻撃の敵に集中します。
4.  **リソースチェック**: メイン武器では対処できない明確な脅威がある場合にのみ、**リコールディスク**や**ジャイロダガー**のようなサイドアームを使用します。
5.  **探索する**: 戦闘が終わった後、すべてのオブジェクトを破壊してジュールジャーや追加のボーンを探します。

| エリアタイプ | 難易度 | 主な危険 | 推奨ツール |
| :--- | :--- | :--- | :--- |
| **オーバーワールド** | 低 | パトロール | ナイトスター |
| **地下墓地** | 中 | 狭い通路 | 潜行 |
| **工業地帯** | 高 | スパイク/炎 | サイドアーム |
| **ボスアリーナ** | 極端 | パターン重複 | 装飾品の相乗効果 |

<Callout type="info" title="探索のヒント">
「適切な壁」とは、まだ学習中のボスです。「悪い壁」とは、ボスにたどり着く前に体力とリソースを使い果たしてしまうルートです。スパークジェネレーターに体力が残っていない状態で到着した場合は、戻ってより効率的なルートを見つけましょう。
</Callout>

</Section>
