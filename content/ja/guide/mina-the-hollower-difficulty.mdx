<Section icon={Shield} title="チャレンジレベルを理解する">

<Section icon={Target} title="スパークシステムとリスク管理">

**Mina the Hollowerの難易度**の核心はスパークシステムにあります。このメカニクスは、進行状況の損失と保持を管理します。死亡が単なるリロードを意味する伝統的なRPGとは異なり、Minaは「ソウルライク」ゲームを彷彿とさせつつ、通貨と経験値の両方の役割を果たす「ボーン」を絡めた独自のひねりを加えたシステムを採用しています。

<CardGrid cols={3}>
  <Card title="スパーク">
    体力の横にある青緑色の点。これを失うと、次の死亡時に所持しているすべてのボーンを失うリスクが生じます。
  </Card>
  <Card title="ボーンの紛失">
    スパークがない状態で死亡すると、ボーンをドロップします。死亡地点に戻り、それらを奪った敵を倒して回収しなければなりません。
  </Card>
  <Card title="回復">
    レベルアップ（ボーニング・アップ）するか、チェックポイントで新しいスパークを購入すると、即座に保護状態が回復します。
  </Card>
</CardGrid>

このシステムは絶え間ない緊張感を生みます。大量のボーンを抱えたまま次のチェックポイントへ突き進むか、それとも引き返してボーンを使い、進行状況を「貯金」するか？ 難易度は持ち運べるスパークの数によっても調整され、ゲーム後半では最大5つまで拡張可能です。

| 死亡ペナルティ | 条件 | 結果 |
| :--- | :--- | :--- |
| **軽度の死** | スパークが有効 | スパークを失うが、すべてのボーンとアイテムを保持 |
| **重度の死** | スパークが無効 | 現在のボーンをすべて紛失；回収が必要 |
| **ツールの紛失** | すべての死 | サブウェポン/サイドウェポンが死亡地点にドロップされる |
| **遺物の状態** | すべての死 | 永続的なアクセサリーは決して失われない |

<Callout type="warning" title="ツールの保持">
アックス（斧）やジャベリン（投げ槍）などのサイドウェポンは強力ですが壊れやすいものです。死亡するとそれらは墓に残されます。回収する前に再び死亡すると、その特定のツールは永遠に失われる可能性があります。
</Callout>

</Section>

<Section icon={Sparkles} title="アクセシビリティと難易度修飾子">

Yacht Club Gamesは、**Mina the Hollowerの難易度**をあらゆるプレイヤーのスキルレベルに合わせられるよう、広範なアクセシビリティ機能を実装しました。オプションメニューには200以上の修飾子が用意されており、ゲームを大幅に簡単にしたり、難しくしたり、あるいは奇妙な体験にしたりと、高度にカスタマイズ可能です。

<Steps>
  <Step num="1" title="アシストモードの有効化">
    アシストモードを有効にすると、被ダメージを軽減し、敵に対するミナの移動速度を向上させることができます。
  </Step>
  <Step num="2" title="ゲームスピードの調整">
    ゲームスピードを微調整できます。例えば、世界の速度を95%に設定しつつミナの速度を100%に維持することで、反応速度において圧倒的な優位を得られます。
  </Step>
  <Step num="3" title="ボーン保護">
    死亡時にボーンを失わないようにする修飾子を切り替えることができ、このジャンルに特有の「死体回収（コープスラン）」のストレスを取り除くことができます。
  </Step>
  <Step num="4" title="チートコード">
    楽しいエフェクトを楽しんだり、特に過酷なプラットフォームセクションをバイパスしたりするために、何百もの内蔵「チート」を利用できます。
  </Step>
</Steps>

これらの修飾子は、単にゲームを簡単にするためのものではありません。ベテランプレイヤーは、世界全体を反転させたり、敵の攻撃性を高めたりして「ハードコア」なプレイ環境を作ることもできます。この柔軟性はゲームデザインの重要な一部であり、高難易度のレトロスタイルゲームにおいて「公平さ」は主観的であることを認めています。

| 修飾子の種類 | ゲームプレイへの影響 | 推奨事項 |
| :--- | :--- | :--- |
| **ダメージスケーリング** | 体力の減少量を増減させる | ボスのパターン学習に使用 |
| **無限スパーク** | ボーンの紛失を完全に防ぐ | カジュアルな探索に最適 |
| **潜りバッファ** | 潜りの受付時間を延長する | タイミングが苦手な場合に不可欠 |
| **アイテムシャッフル** | アイテムの配置をランダム化 | 周回プレイのバリエーションに |

<Callout type="info" title="実績と修飾子">
特定の強力な修飾子やチートを有効にすると、Steamの実績やゲーム内のトロフィーが無効になる場合があります。変更を確定する前に、必ず警告プロンプトを確認してください。
</Callout>

</Section>

<Section icon={Swords} title="戦闘の習熟：武器と潜り">

Mina the Hollowerで成功するには、「潜り（Burrow）」をマスターし、自分のプレイスタイルに合った武器を選ぶことが不可欠です。戦闘は慎重さが求められます。ボタンを連打するだけでは生き残れません。すべての振りには硬直時間があり、すべての敵には突くべき特定の予兆（テル）があります。

**武器のアーキタイプ：**
- **ムチ（Whip）：** 射程とスピードのバランスが取れた「標準的」な選択。
- **フラッシュストライク・モール（Flash Strike Maul）：** 予備動作は極めて遅いが、絶大なダメージとノックバックを与える。
- **ダガー（Daggers）：** 高速で射程が短く、「ガラスの砲台」的なアプローチが必要。
- **サイドウェポン（Sidearms）：** 遠距離オプションを提供する消耗品の投擲物（アックス、ジャベリン）。

潜りメカニクスはミナのシグネチャームーブです。アクションボタンを長押しすることで、ミナは地中に潜ります。これにより、ほとんどの飛び道具や地上ベースの攻撃に対して無敵になります。潜りから地上に出る際にはわずかなスピードブーストが得られ、そのままジャンプ攻撃に移行することも可能です。

| 武器タイプ | スピード | 射程 | 最適な用途 |
| :--- | :--- | :--- | :--- |
| **モール** | 極低速 | 中 | 隙の大きいボス |
| **ムチ** | 中速 | 長 | 一般的な探索と集団戦 |
| **ダガー** | 高速 | 短 | 高機動プレイヤーとスピードラン |
| **アックス（サブ）** | 低速 | 弧状 | 盾持ちや高所にいる敵の撃破 |

<Callout type="success" title="ドロドロ体力の戦略">
遺物「Sludgy Health Vial（ドロドロ体力の瓶）」を装備すると、受けたダメージがゆっくり減少するバーに変化します。バーがゼロになる前に回復すれば、実質的にダメージを無効化でき、本来なら致命的となる攻撃を「耐える」ことが可能になります。
</Callout>

</Section>

<Section icon={Trophy} title="ボーニング・アップ：成長戦略">

「ボーニング・アップ（Boning Up）」と呼ばれるレベルアップシステムは、『リンクの冒険』からインスピレーションを得ています。十分なボーンを集めると、選択を迫られます。経験値を無期限に溜め込むことはできず、激化する**Mina the Hollowerの難易度**を生き抜くためにはキャラクターの成長に投資しなければなりません。

遺物は成長のもう一つの柱です。ゲーム内には60種類以上の遺物があり、「ビルド」を作成できます。例えば、機動力の遺物と飛び道具を反射する遺物を組み合わせれば、遠距離攻撃の敵に対してアグレッシブに立ち回れます。遺物は永続的であり、ミナの能力を最も大きく強化します。

| ステータスカテゴリ | メリット | 戦略的価値 |
| :--- | :--- | :--- |
| **攻撃** | 一撃あたりのダメージ増加 | ボス戦を大幅に短縮 |
| **防御** | 被ダメージの軽減 | ダンジョンでのミス許容度を上げる |
| **サイドウェポン** | ツールの威力/利便性向上 | 遠距離攻撃に頼るプレイヤーに不可欠 |
| **ボーンストーン** | 通貨の安全な保管 | 終盤の武器アップグレードに必須 |

<Callout type="tip" title="質屋の戦術">
ゲームが簡単すぎると感じる場合は、質屋を訪れて初期体力やスパーク、さらには瓶さえも売却できます。「マイナスの達成率」でゲームをクリアすることは、最も熱心なプレイヤー向けの高度な実績です。
</Callout>
