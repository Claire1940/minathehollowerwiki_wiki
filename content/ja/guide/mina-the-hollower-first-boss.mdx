<Section icon={Map} title="オッセクスへの道: ハルクトルーパのミニボス">

最初の主要なボスへの旅は、ローナーズランディングとボードウォークを通ります。オッセクス市に到着する前に、手ごわい門番である**ハルクトルーパ**に直面します。彼は「メイン」の最初のボスではありませんが、準備ができていない多くのプレイヤーにとって難しいと感じる、必須のスキルチェックとして機能します。

ハルクトルーパは巨大な棍棒を使い、2つの主要なフェーズを持っています。彼はしばしばあなたに向かってジャンプして地面を叩きつけたり、チャージされたスイングで前方にテレポートしたりします。アリーナには岩が散らばっています。これらを利用して彼の視線を遮り、重いスイングの衝撃を吸収しましょう。

| 攻撃名 | 説明 | 対処戦略 |
| :--- | :--- | :--- |
| **ジャンプスラッシュ** | 高くジャンプして棍棒を叩きつける | 逃げるか、潜行してAoEを避ける |
| **チャージスイング** | 広い弧を描きながら前方にテレポートする | 岩陰に隠れるか、彼の後ろに回避する |
| **通常攻撃** | 素早い水平方向の薙ぎ払い | 最大武器リーチで突く |

<Callout type="warning" title="まずレベルアップ">
レベル1でハルクトルーパと戦ってはいけません。ボードウォークエリアで時間を費やして**骨**を集め、チェックポイントで「ボーンアップ」しましょう。**攻撃力**をレベル2に上げると、この戦闘が大幅に短縮されます。
</Callout>

<Section icon={Target} title="ソーンの倒し方: 最初のボスガイド">

オッセクスに入り、ラディアントマナーを抜けると、中庭で**ソーン**に直面します。これが正式な**mina the hollower 最初のボス**との遭遇です。ソーンは、これまでに遭遇したどの敵よりも速く、機動性の高いライバル戦士です。彼は槍のような武器を使い、元素の竜巻を操ります。

<Steps>
  <Step num="1" title="影に注目">
    ソーンの最も一般的な動きは、高くジャンプすることです。彼が画面外にいる間は、地面の円形の影に注目してください。これは彼がどこに着地するかを正確に示します。
  </Step>
  <Step num="2" title="衝撃波を回避">
    ソーンが着地すると、紫色の衝撃波を発生させるか、竜巻を召喚します。衝撃地点の近くに立たないでください。効果が消えるのを待ってから移動しましょう。
  </Step>
  <Step num="3" title="2ヒットルール">
    ソーンが着地してAoE攻撃を終えた後、約2秒間静止します。近づいて**2ヒット**当てたら、すぐに後退しましょう。欲張りは、この戦闘で死を招く主な原因です。
  </Step>
  <Step num="4" title="紫色の軌跡を避ける">
    ソーンは時折ダッシュし、紫色の点の軌跡を残します。これらの点は短い遅延の後で爆発します。このフェーズでは移動に完全に集中し、攻撃しようとしないでください。
  </Step>
  <Step num="5" title="元素フェーズ">
    ソーンの体力が50%を下回ると、あなたの動きを追跡する炎の竜巻を召喚します。フェーズが終わるまで、アリーナを大きく円を描くように走り、竜巻を追い越しましょう。
  </Step>
</Steps>

| ソーンの行動 | 危険度 | 最善の対応 |
| :--- | :--- | :--- |
| **ハイジャンプ** | 高 | 影を追跡し、3タイル離れる |
| **炎の竜巻** | 極めて危険 | 円を描くように継続的に移動する |
| **爆発する軌跡** | 中 | 紫色の床マーカーから離れる |
| **槍突き** | 低 | 横にステップして反撃する |

<Callout type="success" title="攻撃的な報酬">
ボスに攻撃を当てると、プラズマゲージが回復することを覚えておいてください。体力が低い場合、成功したコンボを決めることは、逃げるよりも良い選択肢となることが多く、プラズマバイアルをより早く再度使用できるようになります。
</Callout>

</Section>

<Section icon={Trophy} title="高度な戦闘戦術">

**mina the hollower 最初のボス**をマスターすることは、始まりに過ぎません。ゲームの後半を生き残るためには、ミナの潜行能力を攻撃ローテーションに組み込む必要があります。潜行は移動のためだけでなく、短い無敵時間（iフレーム）を付与する防御ツールでもあります。

ソーンやハルクトルーパが広範囲攻撃を行う際、地面に潜行して攻撃をやり過ごすことができます。攻撃判定が消えた直後に地上に出ると、「潜行攻撃」を繰り出すことができ、これは通常攻撃よりも多くのダメージを与えることが多いです。

| メカニクス | 利点 | 最も効果的な対象 |
| :--- | :--- | :--- |
| **潜行回避** | 無敵時間 / 無敵 | 広範囲攻撃や衝撃波 |
| **サブウェポン切り替え** | 多様性 | 飛行する敵 (ダガー) / ボス (斧) |
| **ステータス稼ぎ** | パワースケーリング | 3ヒット以下で倒されるボス |

<Callout type="tip" title="ステータス優先順位">
まず**攻撃力**を優先して戦闘を早く終わらせ、次に**防御力**を上げてミスをしても生き残れるようにしましょう。サブウェポンのステータスには、メイン武器が少なくともレベル4になってから投資しましょう。
</Callout>

</Section>
