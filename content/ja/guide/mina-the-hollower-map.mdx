<Section icon={Target} title="マップの解除方法：救助ミッション">

**mina the hollower map**を解除するには、ホロウワーギルドが関わる特定のイベントシーケンスを完了する必要があります。マップを単に宝箱で見つけることはできません。オスセックス南の危険な地域で行方不明になった仲間のホロウワーを助けることで、マップを獲得する必要があります。

ミッションはホロウワーギルドで始まります。オスセックスのメインエントランスの右側にある石造りの建物です。建物に入るには少しプラットフォームアクションが必要です。建物の右側にある木箱を壊して、中への道を見つける必要があります。中に入ると、ムリエルに会うことができ、レーヌとドリールハルトが焦げた町に消えたことを知らされます。

| 救助要件 | 目的 | 入手方法 |
| :--- | :--- | :--- |
| **1 ケアー** | 焦げた町へのゲートを解除 | オスセックスで見つけるか購入 |
| **プラズマバイアル** | レーヌを癒し、ゲートを開くのに必要 | 標準的な体力回復アイテム |
| **穴掘り能力** | トラップやショートカットをナビゲートするのに必要 | デフォルトのアビリティ |
| **500 ボーン** | マップの購入価格 | 敵からドロップ/壺の中から見つける |

<Steps>
  <Step num="1" title="ホロウワーギルドを見つける">
    オスセックスのメインエントランスの右側へ向かいます。ギルド建物の右側にある箱を壊し、その開口部から飛び込んで中に入ります。ムリエルと話して、行方不明のチームメンバーについて学びます。
  </Step>
  <Step num="2" title="焦げた町に入る">
    オスセックスのメインエントランスから南へ向かいます。**ケアー**を使って北の道の階段を解除します。「1」と書かれた黄色のゲートに遭遇します。プラズマバイアルを1本挿入して、レーヌが捕らえられている複合施設に入場します。
  </Step>
  <Step num="3" title="レーヌを救出し、護衛する">
    罠だらけの部屋をナビゲートして、黄金の狐、レーヌを見つけます。プラズマバイアルを使って彼らを癒します。**注意**: レーヌは穴を掘ることができず、1ブロック幅の隙間しか飛び越えられません。道を切り開き、彼らがついてくるのを待つ必要があります。
  </Step>
  <Step num="4" title="ドリールハルトを見つける">
    レーヌを護衛した後、ホロウワーギルドに戻ります。左下隅の岩から突き出ている緑色の螺旋を探します。それを叩くと、モグラの商人、ドリールハルトが現れます。
  </Step>
</Steps>

<Callout type="warning" title="護衛メカニクス">
レーヌのAIは限られています。あなたが速く動きすぎたり、障害物の下を掘ったりすると、レーヌは置いていかれます。次の画面に進む前に、必ずキツネがついてきているか確認してください。そうしないと、彼らを回収するために引き返す必要があるかもしれません。
</Callout>

<Section icon={Locate} title="マップの購入と閲覧">

ドリールハルトが救出され、ギルドに戻ると、店を開きます。主な目玉商品は、500ボーンで販売される**mina the hollower map**です。これは一度限りの購入で、テネブロス島での残りの旅のためにマッピングシステムを解除します。

マップを購入しても、使用可能なアイテムとしてインベントリには表示されません。代わりに、マップは**アンダーラボ**に物理的に設置されます。このハブエリアの北西の角で見つけることができます。マップディスプレイとインタラクトすると、進行状況を分析できるフルスクリーンインターフェースが表示されます。

| マップの視覚状態 | 意味 | プレイヤーの必要なアクション |
| :--- | :--- | :--- |
| **覆われた雲** | 地域がまだ訪問されていない | エリアを探索する |
| **薄暗い** | 地域は訪問済みだがスパークジェネレーターがオフ | スパークジェネレーターを見つけて修理する |
| **完全に点灯** | スパークジェネレーターがアクティブ | 地域は電力供給のために「クリア」されている |
| **チェックリストアイコン** | 見つかった収集品と不足している収集品を表示 | 100%達成のために使用 |

<Callout type="success" title="マップの場所">
アンダーラボはあなたの主要なハブです。迷ったときはいつでも、ラボの北西の角に戻ってマップを確認してください。どの地域にまだ「暗い雲」が残っているかを確認する最も信頼できる方法です。
</Callout>

<Section icon={Sparkles} title="マップのアップグレードと地域別チェックリスト">

基本的な**mina the hollower map**は、どこに行ったかを示してくれますが、すべてを教えてくれるわけではありません。進行するにつれて、マップを強化して、より詳細な情報を提供することができます。これは、100%コンプリートを目指すプレイヤーにとって特に役立ちます。テネブロス島のすべての地域にチェックリストが追加されるからです。

チェックリスト機能は、壊せる壁や複雑なパズルの後ろに隠されていることが多い特定のアイテムを追跡します。マップをアップグレードすることで、ボーンオーチャードやサンケン・グロットなどのエリアで、あといくつアイテムが不足しているかを正確に確認できます。
