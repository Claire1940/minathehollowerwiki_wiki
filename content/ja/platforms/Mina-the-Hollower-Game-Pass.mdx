<Section icon={Target} title="進行とキャラクターの成長">

Mina the Hollowerは「ボーンアップ」と呼ばれる進行システムを利用しています。ボーンは、エルデンリングのルーンやダークソウルのソウルと同様に、主要な通貨であり経験値です。死亡すると、ボーンは死亡した場所に落ちます。それらを回収する前に再び死亡すると、これらのリソースは永久に失われます。

<Steps>
  <Step num="1" title="ボーンを集める">
    敵を倒し、隠された宝物を見つけてボーンを蓄積します。
  </Step>
  <Step num="2" title="アンダーラボを見つける">
    これらの安全な場所を見つけて、ボーンを恒久的なステータス増加に費やします。
  </Step>
  <Step num="3" title="トリニケットを装備する">
    トリニケットを見つけて交換し、移動、防御、または攻撃力を変更します。
  </Step>
  <Step num="4" title="スパークを管理する">
    スパークを獲得してセーフティネットとして機能させ、死亡時のボーンの損失を防ぎます。
  </Step>
</Steps>

アンダーラボは、体力とフラスコを回復させるだけでなく、世界のすべての敵をリスポーンさせるチェックポイントとして機能します。これにより、次のラボに進むか、現在の獲得物を確保するために撤退するかを決定しなければならない、緊張感のあるゲームプレイループが生まれます。

| ステータス/アップグレード | 効果 | 優先度 |
| :--- | :--- | :--- |
| **攻撃力** | 1ヒットあたりの純粋なダメージを増加させる | 高 |
| **防御力** | 被ダメージを軽減する | 中 |
| **体力拡張** | 体力バーに永続的なHPを追加する | 高 |
| **トリニケットスロット** | より多くのパッシブバフを可能にする | 必須 |

<Callout type="success" title="プロトスパーク">
**プロトスパーク**のトリニケットの入手を優先してください。これはチェックポイント訪問ごとに一度あなたを復活させ、ゲームの難しい23時間のキャンペーン中に文字通りの命の恩人となります。
</Callout>

</Section>

<Section icon={Sparkles} title="探索と世界の秘密">

テネブロス諸島は非線形のオープンワールドです。最初から、4つの初期ダンジョンに好きな順番でアクセスできます。ゲームは特定のアイテムの背後で進行を制限するのではなく、あなたのスキルと秘密の通路を見つける能力が、どこに行けるかを決定します。

**主要な探索ツール:**
- **穴掘り**: 隙間を渡り、地面の小さな穴に入ります。
- **アイアンステッド**: 巨大な裂け目を飛び越えるために使用される自転車型のサイドアーム。
- **トリニケットコンボ**: 壁掴みと穴掘り時間の増加を組み合わせて、「到達できない」宝箱に到達します。
- **画面遷移**: ランバック中に圧倒された場合、敵の位置をリセットするために使用します。

| 地域 | 雰囲気/テーマ | 主要な課題 |
| :--- | :--- | :--- |
| **セプテンバー** | ハロウィン/ゴシックホラー | 潜むバイオハザード風の脅威の管理 |
| **沼地** | 濁った/有毒 | 環境ハザードと遅い移動 |
| **地下墓地** | 暗い/アンデッド | 複雑なパズルと罠だらけの通路 |
| **主要都市** | スチームパンク/テクノロジー | NPCのサイドクエストと高級鍛冶屋 |

<Callout type="info" title="マップの秘密">
Mina the Hollowerには部屋ごとのマップがありません。視覚的な観察に頼る必要があります。壁が怪しいように見えたり、部屋に出口がない場合は、鞭やサイドアームを使って壊せるタイルを探してください。
</Callout>

</Section>
